# 📈 StockOracle — AI 주식 예측 분석기

> 환율 · 유가 · 선물 · 외국인/기관 수급 · 뉴스를 종합 분석해 내일/1주일/1개월 주가를 예측하는 AI 분석 앱

🌐 **라이브 데모**: [https://stock-oracle-opal.vercel.app/](https://stock-oracle-opal.vercel.app/)

> ⚠️ Render 무료 플랜은 비활성 시 슬립 상태가 됩니다. 첫 요청 시 50초가 걸릴 수 있어요.

---

## ✨ 주요 기능

### 📊 데이터 수집 (완전 자동)

- **주가**: 현재가, 52주 고/저, 거래량, 30일 히스토리
- **시장 지표**: 코스피, 달러/원 환율, WTI/브렌트 유가, S&P500·나스닥 선물, 금, 달러/엔
- **외국인/기관 수급**: 최근 10일 일별 순매수/순매도 (네이버 금융 크롤링)
- **뉴스**: 네이버 뉴스(회사 + 한국 시황) + NewsAPI 영문(해외 반응 + 글로벌 이슈) + Groq AI 동적 쿼리 생성으로 총 30~60건
- **기술적 지표**: RSI, MACD, 볼린저밴드, 이동평균(MA5/20/60/120), 스토캐스틱, 거래량 비율
- **재무 지표**: PER/PBR/PSR/EV-EBITDA, ROE/ROA, 영업이익률, 성장률, 부채비율, 배당수익률

### 🤖 AI 분석

- **10개 섹션** 종합 분석 리포트 (마크다운 렌더링)
- **예측 기간 선택**: 내일(1거래일) / 1주일(5거래일) / 1개월(20거래일)
- **주가 방향 예측** (상승/하락/보합) + 예상 등락률 범위
- 사전 신호 집계로 AI 방향 예측 유도 + ATR 기반 등락률 범위 자동 계산
- **보유 포지션 입력 시** 11번째 섹션 자동 추가: 익절·추가매수·손절 전략 (금액 기준)

### 📈 예측 정확도 추적

- 예측 자동 저장 → 다음 영업일 실제 주가와 자동 비교
- 방향 적중률 · 범위 적중률 · 상승/하락별 정확도 분리 집계
- 예측 히스토리 모달 UI

### 🧠 Firebase 학습 메모리 (선택)

- 틀린 예측 패턴을 Firestore에 저장 → 다음 분석 시 프롬프트에 자동 주입
- 과거 실패 패턴과 유사한 상황 감지 시 경고
- 사용자별 완전 격리 (user_id 기반)

### 💬 AI 채팅 (플로팅)

- 분석 완료 후 우하단 플로팅 버튼으로 AI 채팅 접근
- 수집된 데이터 컨텍스트 기반 추가 질문 (손절가, 분할매수 전략 등)

## 🔬 백테스트

과거 날짜 기준으로 AI 예측을 반복 실행해 방향 적중률을 측정합니다. 백엔드 로컬 환경에서 실행합니다.

### 준비

```bash
cd python-backend
source venv/Scripts/activate   # Windows
# source venv/bin/activate     # Mac/Linux
```

`.env` 파일에 `GROQ_API_KEY_1` 이 설정되어 있어야 합니다.

### 실행 예시

```bash
# 삼성전자 — 1월~3월, 주 1회, 1주일 예측
python backtest.py \
  --ticker 005930.KS \
  --name 삼성전자 \
  --country 한국 \
  --start 2026-01-06 \
  --end 2026-03-21 \
  --period week \
  --interval 5 \
  --sleep 5

# 애플 — 1월~3월, 매일, 내일 예측
python backtest.py \
  --ticker AAPL \
  --name Apple \
  --country 미국 \
  --start 2026-01-06 \
  --end 2026-03-21 \
  --period tomorrow \
  --interval 1 \
  --sleep 5
```

### 옵션 설명

| 옵션          | 설명                                  | 기본값   |
| ------------- | ------------------------------------- | -------- |
| `--ticker`    | 티커 심볼                             | 필수     |
| `--name`      | 회사명                                | 필수     |
| `--country`   | 한국 / 미국 / 일본                    | 한국     |
| `--start`     | 시작일 (YYYY-MM-DD)                   | 필수     |
| `--end`       | 종료일 (YYYY-MM-DD)                   | 필수     |
| `--period`    | tomorrow / week / month               | tomorrow |
| `--interval`  | 테스트 간격 (영업일)                  | 1        |
| `--sleep`     | AI 호출 간 대기(초) — rate limit 방지 | 3        |
| `--no-ai`     | AI 호출 없이 데이터 수집만 테스트     | -        |
| `--no-review` | 틀린 예측 자기분석 생략               | -        |
